# svga增加定制key、文字、图片工具
> 让设计减少杂事专注创作，减轻运营等待排期压力

### svga加key（仅适用于头饰、勋章等轻量级的文件）
* 拖动key值范围新增透明图层，命名根据开发设定要求即可
* 常用key值命名在缓存中保留
* 导出即可使用（flutter语言可能需要二次转换一下）

### svga加字（仅适用于头饰、勋章等轻量级的文件）
* 在画布中拖动范围新增透明图层，修改内容文案
* 可设置字体、渐变、字号大小
* 可设置动画，动画可多选
* 导出即可使用（flutter语言可能需要二次转换一下）
* 可选取某一帧导出缩略图

### svga加图片（仅适用于头饰、勋章等轻量级的文件）
* 在画布中拖动范围新增透明图层，添加图片
* 可设置动画，动画可多选
* 导出即可使用（flutter语言可能需要二次转换一下）
* 可选取某一帧导出缩略图

可框选多个，点击不同区域修改对应图层内容
